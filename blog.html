1.What is different among map, forEach, filter and find?

Ans: There are some different in map, filter, forEach and find in javascript. The difference are given below.
map: In javascript map() use for loop a array and return a new array.
forEach: In javascript forEach() use for loop a array but it does not return a new array.
filter: In javascript filter() use to get the value depend on condition.filter() return an array with those value are true for the given condition.
find: In javascript find() give the first value for given condition true. It only give the first value.
